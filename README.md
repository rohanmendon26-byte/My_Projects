# 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/rohanmendon26-byte/My_Projects.git
```

Navigate to the project folder:

```bash
cd My_Projects
```

Open any project folder and run it locally.

For React projects:

```bash
npm install
npm run dev
```
